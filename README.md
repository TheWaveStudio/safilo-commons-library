# Safilo Commons

Common modules for Safilo e-commerce (i.e. Carrera, Polaroid).


## Getting started


### Install
In order to install the dependency

**NPM**
```
npm install --save
```

**Yarn**
```
yarn add 
```

**.dependencies**
```
bootstrap:~>4.6.0,
oruga: Vue3 version,
```

** STORYBOOK **
```
Carrera: yarn storybook-cr / npm run storybook-cr
```
